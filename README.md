# Data Governance – Practical Work (with GenAI support)

This repository contains **educational material** for teaching and practicing Data Governance concepts, including metadata management, data quality, privacy/PII classification, and minimal governance controls.

The material is designed for **graduate-level or professional courses** and can be used with or without Generative AI tools.

---

## Scope

The repository includes:
- A synthetic transactional dataset
- A practical assignment (TP) with evaluation rubric
- Reference documents for Data Governance, PII classification, and BIAN (light)
- An example of expected analytical output (summary level)

The focus is **governance thinking and decision-making**, not tooling.

---

## Disclaimer

- All data included is **synthetic** and created solely for educational purposes.
- This repository does **not** represent real customer data, internal policies, or practices of any financial institution.
- BIAN is used as a **conceptual reference**, not as a mandatory or exhaustive standard.

---

## Repository Structure

- `docs/`  
  Reference material used as conceptual grounding.
- `datasets/`  
  Synthetic datasets for exercises and assignments.
- `tp/`  
  Practical assignment description and grading rubric.
- `examples/`  
  High-level example outputs (for instructors).

---

## Intended Audience

- Data Governance students
- Data stewards and data owners in training
- Analytics and data professionals transitioning to governance roles
